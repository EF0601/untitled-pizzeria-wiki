Content

Codes are starter packs for your game. Each game ONLY permits one code to be used. After one code is used, the codes input is locked and has a gray underline.
Active codes can be found occasionally after each release, usually hidden in the notes and will generally expire after several releases. However, two codes are open forever:

  • "Enter code for THIS game." (+100 coins; +5 pizzas) A general easter egg.
  • "0.2.0-alpha" (+40 coins; +2 pizzas, +4 dough, +4 toppings, +4 sauce) A tribute to the beginnings of codes.
